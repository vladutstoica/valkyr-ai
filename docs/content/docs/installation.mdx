## Download Valkyr

### macOS

The easiest way to install on macOS is via Homebrew:

```bash
brew install --cask valkyr
```

Or download directly:

- [Apple Silicon (arm64)](https://github.com/generalaction/valkyr-ai/releases/latest/download/valkyr-arm64.dmg)
- [Intel (x64)](https://github.com/generalaction/valkyr-ai/releases/latest/download/valkyr-x64.dmg)

### Windows

Download the installer or portable version:

- [Installer (x64)](https://github.com/generalaction/valkyr-ai/releases/latest/download/valkyr-x64-installer.exe)
- [Portable (x64)](https://github.com/generalaction/valkyr-ai/releases/latest/download/valkyr-x64.exe)

### Linux

Choose your preferred format:

- [AppImage (x64)](https://github.com/generalaction/valkyr-ai/releases/latest/download/valkyr-x64.AppImage)
- [Debian package (x64)](https://github.com/generalaction/valkyr-ai/releases/latest/download/valkyr-x64.deb)

Or run via Nix:

```bash
nix run github:generalaction/valkyr-ai
```

## Setup

### 1. Install a CLI provider

Valkyr runs coding agents through their CLI. Install at least one [provider](/providers) before getting started:

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

### 2. Connect GitHub

To create PRs (and use GitHub Issues), connect your account in **Settings â†’ Integrations**. Valkyr works with OAuth or authentication via [GitHub CLI](https://cli.github.com):

```bash

```
