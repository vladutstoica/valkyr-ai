## Release Process

Valkyr follows [Semantic Versioning](https://semver.org/):

- **Patch versions** (0.0.x): Bug fixes and minor improvements
- **Minor versions** (0.x.0): New features and capabilities
- **Major versions** (x.0.0): Breaking changes
